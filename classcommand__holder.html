Detailed Description

template<typename proxy>
class command_holder< proxy >

Command holder.

This class can handle, register and execute commands with corresponding callbacks. Command is std::string, callback needs to be "void callback(const std::string &worker_identity, const std::vector<std::string> &args, const command_context<proxy> &context". proxy is templated argument, which specifies type of used communication class.
